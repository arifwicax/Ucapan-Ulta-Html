Ucapan Ulang Tahun

Deskripsi Proyek

Fitur Utama

- Tampilan jam dan tanggal real-time
- Notifikasi WhatsApp yang interaktif
- Animasi typing effect untuk pesan
- Efek salju animasi
- Background musik
- Ayat Al-Quran (QS. Luqman: 12)
- Doa-doa untuk penerima pesan
- Tombol konfirmasi via WhatsApp
- Desain responsive untuk mobile dan desktop

Teknologi yang Digunakan

- HTML5
- CSS3 dengan animasi
- JavaScript vanilla
- SweetAlert2 untuk dialog interaktif
- Google Fonts (Playfair Display, Poppins, Amiri)
- Material Icons


Cara Menggunakan

1. Buka file index.html di browser
2. Tunggu loading screen selesai
3. Klik notifikasi WhatsApp yang muncul
4. Masukkan nama Anda
5. Klik tombol "Buka Pesannya" untuk melihat pesan lengkap
6. Scroll ke bawah untuk membaca semua pesan
7. Klik tombol WhatsApp di akhir untuk mengirim konfirmasi

Desain

Halaman ini menggunakan gradient biru-ungu sebagai background dengan elemen-elemen berikut:
- Card notifikasi dengan efek blur
- Animasi slide-in dan fade-in
- Efek hover interaktif
- Typography yang elegan dengan font Amiri untuk teks Arab
- Animasi salju yang muncul di akhir
